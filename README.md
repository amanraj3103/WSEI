# WSEI Academy

Full-stack student management platform built for Akademia WSEI. Provides admin and student portals with secure authentication, profile management, and administrative controls.

## What it does

- Admin dashboard with student search, filtering, sorting, and status management (Active/Inactive/Graduated/Suspended)
- Student portal for viewing and managing personal profiles
- Auto-generated student email addresses (firstname.lastname@student.wsei.pl) and registration numbers
- Secure authentication with bcrypt password hashing
- RESTful API endpoints for all CRUD operations
- Statistics dashboard showing enrollment metrics
- Duplicate detection to prevent duplicate student records

## Tech stack

Next.js 14, TypeScript, PostgreSQL, Prisma ORM, NextAuth.js, Tailwind CSS, bcryptjs, Vercel
